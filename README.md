# AI-Powered Personalized Learning Assistant (SDG 4)

## Project Overview
This project is an AI-based learning assistant designed to provide personalized explanations, examples, and quizzes based on a learner’s level.

## SDG Alignment
UN Sustainable Development Goal 4 – Quality Education

## Features
- Adaptive learning (Beginner to Expert)
- AI-generated explanations
- Real-world examples
- Automated quizzes with answers

## Technologies Used
- Python
- Google Colab
- Google Gemini API
- Prompt Engineering

## How to Run
1. Open the notebook in Google Colab
2. Add your own API key
3. Run all cells
4. Enter topic and learner level

## Ethical Considerations
- No personal data is collected
- AI is used only for educational purposes
